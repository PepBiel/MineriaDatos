# Minería de Datos

Repositorio académico con varios proyectos de **Data Mining en R**, organizados por carpetas. Cada carpeta corresponde a una práctica independiente, con su propio objetivo, dataset, metodología y autoría.

## Estructura del repositorio

- `Assignment1` — Análisis exploratorio y estadística descriptiva manual sobre `ChickWeight`.
- `Assignment2` — Modelos de regresión sobre el dataset `Wage` del paquete `ISLR`.
- `Assignment3` — Clasificación de aprobación de préstamos con varias técnicas supervisadas.
- `Assignment4` — Clustering sobre datos de pingüinos con métodos jerárquicos, particionales y basados en densidad.
- `Assignment5` — Minería de reglas de asociación con `Apriori` y `Eclat` sobre transacciones de panadería.
- `FinalPractice` — Análisis y modelado del mercado inmobiliario en Portugal para detectar propiedades infravaloradas.

## Tecnologías principales

- **R / R Markdown**
- Librerías como `ggplot2`, `dplyr`, `caret`, `rpart`, `arules`, `dbscan`, `mclust`, `factoextra`, `ISLR`, entre otras
- Entregables en **`.Rmd`**, **`.html`** y en algunos casos datasets `.csv`

## Cómo revisar los proyectos

Cada carpeta contiene al menos un fichero **`.Rmd`** y su versión renderizada en **`.html`**. La forma más sencilla de revisarlos es:

1. Abrir el `.html` para ver el resultado final.
2. Abrir el `.Rmd` si quieres revisar el proceso completo, el código y la interpretación.
3. En los proyectos con datasets incluidos, mantener la misma estructura de carpetas para poder volver a ejecutar el análisis.

## Valor del repositorio

Este conjunto de proyectos muestra una progresión bastante completa dentro de la minería de datos:

- exploración y visualización,
- estadística descriptiva,
- regresión,
- clasificación,
- clustering,
- reglas de asociación,
- y análisis aplicado a un problema de negocio real.

Es un repositorio útil para portfolio porque refleja tanto **base estadística** como **capacidad de aplicar modelos a problemas distintos**.
